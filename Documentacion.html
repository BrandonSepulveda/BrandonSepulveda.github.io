Inicio

El menú de Inicio proporciona acceso rápido a funciones básicas del sistema y herramientas de diagnóstico.

Panel de Control

Abre el Panel de Control clásico de Windows para configurar y personalizar el sistema operativo.

control.exe

Diagnóstico Rápido

Realiza un análisis rápido del sistema para identificar problemas críticos del hardware y software.

Elementos analizados:
  • Espacio disponible en disco (alerta si menos del 10%)
  • Uso de memoria RAM (alerta si más del 80%)
  • Estado de servicios críticos de Windows
  • Actualizaciones pendientes
  • Eventos críticos recientes en el registro de Windows

Ajustes de Windows

Abre la aplicación de Configuración moderna de Windows.

Start-Process "ms-settings:"

Explorador de Archivos

Inicia el Explorador de archivos de Windows.

explorer.exe

PowerShell

Abre PowerShell, la interfaz de línea de comandos avanzada de Windows.

Nota:

Intenta abrir PowerShell 7 si está disponible. Si no, abre PowerShell 5 (integrado en Windows).

Información del Sistema

Abre la herramienta de información del sistema integrada en Windows.

msinfo32.exe

Instalar PowerShell 7

Instala la última versión de PowerShell 7, una versión mejorada con más características y mejor rendimiento.

Nota:

Se instala automáticamente usando Winget si está disponible. Si no está disponible Winget, se descarga e instala el MSI desde GitHub.

Información del Sistema

Información Completa del Sistema

Proporciona un informe detallado sobre el hardware y software del equipo.

Información proporcionada:
  • Sistema operativo: Versión de Windows, compilación, arquitectura, fecha de instalación
  • Procesador: Modelo, núcleos, hilos, velocidad máxima
  • Memoria RAM: Capacidad total, detalles de módulos, fabricante, velocidad
  • Almacenamiento: Capacidad, espacio libre, tipo (SSD/HDD), interfaces
  • GPU: Modelo, memoria dedicada, resolución actual, versión del controlador
  • Placa base: Fabricante, modelo
  • Red: Adaptadores activos, direcciones IP, MAC, velocidad de conexión
  • Seguridad: Estado de Windows Defender, Firewall, BitLocker
  • Batería: Estado de carga, capacidad, vida estimada (para portátiles)
  • Monitores: Modelos conectados, información de fabricante
  • Rendimiento actual: Uso de CPU, memoria y disco en tiempo real

Descargas Apps

Instalador de Aplicaciones

Interfaz gráfica para instalar aplicaciones populares utilizando Chocolatey o Winget.

Características:
  • Selección entre gestores de paquetes Chocolatey o Winget
  • Instalación automatizada del gestor de paquetes si no está disponible
  • Búsqueda de aplicaciones con filtrado en tiempo real
  • Selección múltiple para instalar varias aplicaciones a la vez
  • Visualización del progreso de instalación
  • Registro detallado del proceso
Aplicaciones disponibles:

7zip, aida64, anydesk, brave, cpu-z, crystaldiskinfo, dellcommandupdate, discord, googlechrome, hp-pc-hardware-diagnostics, hpimageassistant, hpsmart, hpsupportassistant, intel-driver-support-assistant, lenovovantage, lenovosystemupdate, microsoft365uninstall, microsoftpowertoys, myasus, notepadplusplus, nzxt-cam, powershell, rufus, teamviewer, ventoy, visualstudiocode, whatsapp, windowsterminal

Reparación

Crear Punto de Restauración

Crea un punto de restauración del sistema antes de aplicar cambios importantes.

Checkpoint-Computer -Description "Punto de restauracion creado por ToolboxBS" -RestorePointType "MODIFY_SETTINGS"
Nota:

Permite revertir el sistema a un estado anterior en caso de problemas con actualizaciones o cambios en la configuración.

Mantenimiento del S.O.

Realiza tareas de mantenimiento avanzadas en el sistema operativo Windows.

bcdedit /set useplatformtick yes
bcdedit /set disabledynamictick yes
sfc /scannow
DISM.exe /Online /Cleanup-image /CheckHealth
DISM.exe /Online /Cleanup-image /Restorehealth
sfc /scannow
Dism.exe /Online /Cleanup-Image /startComponentCleanup
Tareas realizadas:
  • Configuración del temporizador del sistema para mejor rendimiento
  • Verificación y reparación de archivos del sistema (SFC)
  • Diagnóstico y reparación de la imagen de Windows (DISM)
  • Limpieza de componentes de Windows Update

Información de Pantallazos Azules

Analiza y proporciona información sobre los pantallazos azules (BSOD) que han ocurrido en el sistema.

Opciones disponibles:
  • BlueScreenView: Herramienta para visualizar detalles de los BSODs
  • Script Remoto: Análisis avanzado de las causas comunes de BSODs

Ver Estado de la Batería

Proporciona información detallada sobre el estado y salud de la batería en dispositivos portátiles.

powercfg /batteryreport /output "C:\battery_report.html"
Opciones disponibles:
  • BatteryInfoView: Herramienta para detalles avanzados de la batería
  • Informe de Batería: Genera un informe HTML completo con información histórica

Información de Drivers

Proporciona información detallada sobre los controladores instalados en el sistema.

Opciones disponibles:
  • Exportar Drivers: Guarda la información de controladores en un archivo de texto
  • Administrador de Dispositivos: Abre la herramienta de gestión de hardware de Windows

Reparar Red

Soluciona problemas comunes de conexión a Internet y red.

netsh winsock reset
netsh int ip reset
ipconfig /release
ipconfig /renew
ipconfig /flushdns
Operaciones realizadas:
  • Restablecer catálogo Winsock
  • Restablecer TCP/IP stack
  • Liberar y renovar la dirección IP
  • Limpiar la caché DNS

Limpieza

Herramientas para liberar espacio y optimizar recursos del sistema.

Opciones disponibles:
  • Limpiar Archivos Temporales: Elimina archivos temporales del sistema y del usuario
  • Optimizar RAM: Libera memoria utilizando la herramienta RAMMap de Sysinternals

Tweaks

Ejecutar WinUtil

Ejecuta la herramienta WinUtil de Chris Titus Tech para optimizaciones avanzadas del sistema.

powershell -NoExit -Command "irm christitus.com/win | iex"
Características:
  • Optimizaciones de rendimiento y privacidad
  • Instalación y desinstalación de aplicaciones
  • Configuración de opciones avanzadas de Windows
  • Mejoras de seguridad
  • Activación de características ocultas

Instalar Winget

Instala el gestor de paquetes oficial de Windows (Windows Package Manager).

Invoke-WebRequest -Uri https://aka.ms/getwinget -OutFile Microsoft.DesktopAppInstaller_8wekyb3d8bbwe.msixbundle
Invoke-WebRequest -Uri https://aka.ms/Microsoft.VCLibs.x64.14.00.Desktop.appx -OutFile Microsoft.VCLibs.x64.14.00.Desktop.appx
Invoke-WebRequest -Uri https://github.com/microsoft/microsoft-ui-xaml/releases/download/v2.7.3/Microsoft.UI.Xaml.2.7.x64.appx -OutFile Microsoft.UI.Xaml.2.7.x64.appx
Add-AppxPackage Microsoft.VCLibs.x64.14.00.Desktop.appx
Add-AppxPackage Microsoft.UI.Xaml.2.7.x64.appx
Add-AppxPackage Microsoft.DesktopAppInstaller_8wekyb3d8bbwe.msixbundle
Nota:

Winget facilita la instalación y actualización de aplicaciones desde la línea de comandos. Es el gestor de paquetes nativo de Microsoft para Windows 10 y 11.

Activar Windows

Activa Windows utilizando la clave digital integrada en la BIOS/UEFI del sistema.

# Obtener la clave digital almacenada en la BIOS
$clave = (Get-WmiObject SoftwareLicensingService | Select-Object -ExpandProperty OA3xOriginalProductKey)

# Establecer la clave como predefinida
slmgr.vbs /ipk $clave
Nota:

Esta función recupera la clave digital de Windows que está almacenada en la BIOS de fábrica (si está disponible) y la aplica automáticamente.

Ejecutar Optimizer

Descarga e inicia Optimizer, una herramienta de optimización completa para Windows.

Características de Optimizer:
  • Limpieza del sistema
  • Gestión de servicios
  • Eliminación de telemetría
  • Opciones de privacidad
  • Gestión de aplicaciones de inicio
  • Personalización del sistema

Buscar Drivers

Identifica el modelo y número de serie del equipo y busca los controladores correspondientes en línea.

# Obtener información del equipo
$modelo = (Get-WmiObject -Class:Win32_ComputerSystem).Model
$serial = (Get-WmiObject -Class:Win32_BIOS).SerialNumber

# Buscar drivers para este modelo específico
Start-Process "https://www.google.com/search?q=$modelo+$serial+drivers"

Actualizar Windows

Inicia el proceso de búsqueda e instalación de actualizaciones de Windows.

Start-Process -FilePath "C:\Windows\System32\UsoClient.exe" -ArgumentList "StartInteractiveScan" -NoNewWindow
Nota:

Este comando inicia el escáner de Windows Update de forma interactiva, permitiendo al usuario elegir qué actualizaciones instalar.

Analizar y Optimizar S.O.

Ejecuta un script personalizado que analiza y optimiza el sistema operativo Windows.

powershell.exe -NoProfile -ExecutionPolicy Bypass -Command "irm 'https://raw.githubusercontent.com/BrandonSepulveda/ToolboxBS/refs/heads/main/procesos/Analizador.ps1' | iex"
Funciones del analizador:
  • Diagnóstico completo del sistema
  • Identificación de problemas y cuellos de botella
  • Sugerencias personalizadas de optimización
  • Aplicación automática de mejoras recomendadas

Propiedades del Sistema

Abre la ventana de propiedades del sistema para ajustar la configuración de rendimiento y otras opciones avanzadas.

control.exe sysdm.cpl
Nota:

Permite acceder a configuraciones avanzadas como opciones de rendimiento, variables de entorno, inicio y recuperación, y protección del sistema.

Deshabilitar Transparencia

Desactiva los efectos de transparencia en Windows para mejorar el rendimiento visual.

$registryPath = "HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize"
$name = "EnableTransparency"
$value = 0
Set-ItemProperty -Path $registryPath -Name $name -Value $value
Nota:

Deshabilitar los efectos de transparencia puede mejorar el rendimiento en equipos con hardware limitado.

Apagar y Entrar a BIOS

Reinicia el sistema y entra directamente a la configuración del BIOS/UEFI.

Start-Process -FilePath "shutdown" -ArgumentList "/r /fw /t 1" -Verb RunAs
Nota:

Este comando permite acceder a la configuración de la BIOS sin necesidad de presionar teclas especiales durante el inicio del sistema.

Entorno de Recuperación

Reinicia el sistema y entra al entorno de recuperación de Windows.

shutdown /r /o /f /t 00
Opciones disponibles en el entorno de recuperación:
  • Restaurar a un punto anterior
  • Restablecer el PC
  • Iniciar en modo seguro
  • Símbolo del sistema para solucionar problemas avanzados
  • Restaurar desde una imagen del sistema

Actualizar Apps

Actualiza todas las aplicaciones instaladas mediante Winget.

winget update --all
Nota:

Este comando busca actualizaciones para todas las aplicaciones instaladas a través de Winget y las actualiza automáticamente.

Herramientas

Ejecutar ToolboxBS

Inicia la última versión de ToolboxBS directamente desde GitHub.

irm "https://brandonsepulveda.github.io/tbs" | iex
Nota:

Este comando descarga e inicia la última versión de ToolboxBS sin necesidad de instalación.

Ejecutar WinScript

Ejecuta WinScript, una herramienta complementaria para optimización de Windows.

irm https://winscript.cc/irm | iex

Instalar Sentinel

Descarga e instala Hard Disk Sentinel para monitorear la salud de los discos duros.

Características:
  • Monitoreo de la salud de discos duros y SSD
  • Detección temprana de fallos
  • Información detallada de temperatura
  • Pruebas de rendimiento
  • Alertas preventivas

Optimizar Sistema

Aplica una serie de optimizaciones avanzadas al sistema operativo para mejorar el rendimiento.

Optimizaciones aplicadas:
  • Creación de un punto de restauración
  • Deshabilitación de telemetría y rastreo
  • Optimización de servicios para mejor rendimiento
  • Activación del modo de rendimiento definitivo
  • Aplicación de ajustes para videojuegos
  • Personalización de la interfaz de Windows

Instalar Intel Driver Support

Descarga e instala Intel Driver Support Assistant para la gestión automatizada de controladores Intel.

Características:
  • Detección automática de hardware Intel
  • Verificación de controladores desactualizados
  • Descarga e instalación de los controladores más recientes
  • Soporte para gráficas, chipsets, redes y más

Reparación de Windows

Ejecuta comandos avanzados de reparación del sistema operativo Windows.

sfc /scannow
DISM.exe /Online /Cleanup-image /Restorehealth
Operaciones realizadas:
  • Verificación de la integridad de todos los archivos protegidos del sistema
  • Reparación de archivos dañados
  • Reparación de la imagen del sistema operativo

Administrador de Tareas

Abre el Administrador de tareas de Windows para monitorear y gestionar procesos y rendimiento.

taskmgr.exe

Administrador de Dispositivos

Abre el Administrador de dispositivos para visualizar y gestionar el hardware del sistema.

devmgmt.msc

Servicios

Abre la consola de administración de servicios de Windows.

services.msc
Nota:

Permite iniciar, detener, habilitar o deshabilitar los servicios del sistema. Es una herramienta avanzada para usuarios con conocimientos técnicos.

Limpieza de Disco

Inicia la herramienta de limpieza de disco integrada en Windows.

cleanmgr.exe
Nota:

Permite eliminar archivos temporales, caché del navegador, archivos de la papelera y otras categorías de archivos innecesarios para liberar espacio en disco.

Editor del Registro

Abre el Editor del registro de Windows para realizar modificaciones avanzadas.

regedit.exe
Precaución:

El Editor del registro es una herramienta avanzada. Los cambios incorrectos pueden causar problemas graves en el sistema. Se recomienda crear un punto de restauración antes de realizar cambios.

Verificar Actualizaciones

Abre la configuración de Windows Update para buscar e instalar actualizaciones.

Start-Process "ms-settings:windowsupdate"
